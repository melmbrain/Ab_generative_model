### Journal-Specific Requirements

#### For Bioinformatics

- [ ] Format according to journal guidelines
  - [ ] Maximum 15 pages including figures
  - [ ] Figures at end (not embedded)
  - [ ] Tables in main text or supplementary
  - [ ] Vancouver reference style

- [ ] Additional requirements
  - [ ] Graphical abstract (optional)
  - [ ] Author photo (optional)
  - [ ] Preprint DOI (if posting to bioRxiv)

#### For BMC Bioinformatics

- [ ] Format according to journal guidelines
  - [ ] Structured abstract required
  - [ ] Keywords (3-10)
  - [ ] Abbreviations list
  - [ ] Vancouver or Harvard reference style

- [ ] Additional requirements
  - [ ] Open access fee ($2,500 - check waivers)
  - [ ] ARRIVE guidelines (N/A for computational)
  - [ ] Preprint encouraged

#### For Nature Methods

- [ ] Format according to journal guidelines
  - [ ] Summary paragraph (150 words)
  - [ ] Main text 3,000 words
  - [ ] Methods 3,000 words
  - [ ] Nature reference style

- [ ] Additional requirements
  - [ ] Code availability (GitHub)
  - [ ] Reproducibility checklist
  - [ ] Data availability (Zenodo)
  - [ ] Reporting summary

| Phase | Duration | Tasks |
|-------|----------|-------|

| **Decision** | 1-2 weeks | Accept/reject decision |
| **Production** | 2-4 weeks | Proofs, final edits |
| **Publication** | Day 0! | Paper goes live |

**Total timeline**: 2-4 months (computational journal)
